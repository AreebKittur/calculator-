# JavaScript Calculator

## Description

This is a simple and responsive **JavaScript Calculator** built using **HTML, CSS, and JavaScript**. The application performs basic arithmetic operations through an easy-to-use interface. Users can enter numbers and operators using either the on-screen buttons or their keyboard, making calculations quick and convenient.

The calculator supports addition, subtraction, multiplication, division, percentage calculations, and decimal numbers. It also includes useful features such as **All Clear (AC)** to reset the display, **Backspace (⌫)** to delete the last entered character, and keyboard shortcuts for improved usability. When the **"="** button or the **Enter** key is pressed, the calculator evaluates the entered expression and displays the result.

The project demonstrates the fundamentals of front-end web development, including HTML for page structure, CSS for styling and responsive design, and JavaScript for implementing calculator logic, event handling, and DOM manipulation.

## Features

* Basic arithmetic operations (+, −, ×, ÷)
* Percentage (%) calculations
* Decimal number support
* All Clear (AC) button
* Backspace (⌫) functionality
* Keyboard input support
* Responsive and modern user interface
* Fast and simple calculations

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Start performing calculations.
